RetireWise

Retirement Account Dashboard · Full-Stack
Overview

A full-stack retirement account dashboard for tracking 401(k) balances, monitoring contribution history, and generating retirement income projections. Built with Angular 19 on the frontend and Spring Boot 3 on the backend, with JWT authentication, role-based access control, and interactive Chart.js visualizations.

Features
Dashboard Overview
Total account balance, YTD contribution progress against IRS limits, balance trend line chart, and asset allocation doughnut chart.
Contribution Management
Paginated contribution history with employee/employer breakdown, modal for adding new contributions, and pay period tracking.
Retirement Projections
Compound growth calculator with customizable parameters — projected future balance and estimated monthly retirement income using the 4% withdrawal rule.
Authentication & Security
Stateless JWT sessions with refresh tokens, BCrypt password hashing, role-based access control, XSS/clickjacking protection headers, and CORS configuration.
Data Visualization
Interactive Chart.js charts for balance trends, asset allocation breakdowns, and growth projections with customizable parameters.
Database Migrations
Flyway-managed schema migrations with PostgreSQL 16, JPA/JDBC persistence layer, and Docker Compose for local development.
Highlights

What I Built

  • Designed and built the Angular 19 SPA frontend with TypeScript — dashboard views, contribution management, and retirement projection calculators
  • Implemented interactive Chart.js visualizations for balance trends, asset allocation doughnut charts, and compound growth projections
  • Built the Spring Boot 3 REST API with Java 17 — account endpoints, contribution CRUD, and projection calculations
  • Engineered JWT authentication with access + refresh tokens, BCrypt hashing, and role-based access control via Spring Security
  • Set up PostgreSQL 16 with Flyway migrations, Docker Compose for local dev, and GitHub Actions CI/CD pipeline
  • Applied security hardening — XSS/clickjacking headers, input validation via Jakarta Bean Validation, and CORS configuration
