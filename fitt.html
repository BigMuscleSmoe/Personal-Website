Features
Workout Logging
Exercise logging modals with add/remove functionality, 90-second countdown rest timer with useRef intervals, and per-exercise tracking.
Nutrition Tracking
Macronutrient breakdowns and daily calorie summaries with food search powered by Open Food Facts API.
Progress Analytics
4 interactive Chart.js visualizations — activity trends, sleep quality, training volume, and lift-specific progress via react-chartjs-2.
AI Coaching Chat
Conversational interface with typing indicator animations and markdown rendering for personalized fitness and nutrition coaching powered by Groq.
Recovery Insights
Sleep and fatigue signal tracking that adjusts training recommendations based on recovery data.
Auth & Security
Supabase JWT authentication with structured REST API consumption from the FastAPI/Python backend.
My Role — Frontend Lead

What I Built

  • Architected and built the complete frontend (~3,700 lines of React/JSX + CSS) for a mobile-first fitness web app
  • Implemented 4 interactive Chart.js visualizations (activity trends, sleep quality, training volume, lift-specific progress) using react-chartjs-2
  • Designed a custom dark-theme CSS design system (~980 lines) with responsive layouts, consistent color tokens, and smooth animations optimized for mobile viewports
  • Built a 90-second countdown rest timer, exercise logging modals, and a nutrition logging UI with macronutrient breakdowns
  • Developed the AI chat interface with typing indicator animations and markdown rendering support
  • Integrated Supabase JWT authentication and structured the frontend for REST API consumption with the FastAPI backend
  • Collaborated via GitHub pull requests and branch-based workflow with backend, database, and AI/ML contributors
