TCP List Manager

CSE 3421 · Intro to Computer Networking
Overview

A Python TCP client-server application that manages named lists over a persistent socket connection using a JSON-based protocol. The client validates input locally before transmitting commands; the server processes them, maintains state in memory, and persists active lists to disk on shutdown. Built for CSE 3421 at Ohio State.

How it works

The real version uses TCP sockets for communication between two separate Python processes. This browser demo simulates the same JSON request/response protocol entirely in JavaScript — same validation rules, same error handling, same server logic. No network involved.
