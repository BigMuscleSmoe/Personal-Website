Core Language Interpreter

Compiler & Interpreter Fundamentals
Overview

A full interpreter for the Core programming language, built in Java using a four-stage pipeline: tokenization, recursive-descent parsing, AST construction, and tree-walk execution. Supports integer arithmetic, conditionals, loops, and I/O — with pretty-printing and detailed error reporting at every stage.

How it works

The real interpreter is written in Java with separate classes for each AST node — tokenizer, parser, and executor. This browser demo reimplements the same four-stage pipeline entirely in JavaScript: tokenization, recursive-descent parsing, AST pretty-printing, and tree-walk execution — same grammar rules, same error messages.
